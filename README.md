# PowerShell-Ciberseguridad
Equipo de trabajo
¿Qué contiene?
El proyecto contiene un script en PowerShell diseñado para realizar escaneos básicos de seguridad en la red del equipo, tambien identifica las direcciones IP externas con las que la computadora tiene una comunicación activa, analiza si las IP tienen reportes de actividad maliciosa y genera un reporte de cada una, tambien guarda los resultados en un archivo CSV en el escritorio del usuario 

¿Qué scripts se incluyen?
Get-ComunicacionIPs
Investigar-IP
Get-Reporte
Guardar-Reporte
Script principal de ejecución 

¿Qué tareas de ciberseguridad resuelven?
identifica conexiones externas activas del equipo
detecta posibles IP maliciosas con las que el sistema esté comunicandose, Consulta informacion de reputación de direcciones IP usando una base de datos de amenazas, Clasifica las Ip según su nivel de riesgo, Genera reportes de  seguridad automatizados para análisis posterior

¿Qué aprendiste al desarrollarlos?
Aprendi a obtener conexiones de red activas, a filtrar direcciones IP internas y externas, a generar reportes en formato CSV y aprendi la importancia de automatizar tareas de monitoreo de seguridad ya que eso nos ayuda a detectar posibles riesgos de forna rapida.
